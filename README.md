游戏主流程
预载页面（按钮进入）->菜单栏->选择开始游戏后->进入过场的ui场景->按下任意键->Timeline制作的过场动画->进入角色操作的场景->与石碑交互->进入关卡->每关卡结束会随机发放卡牌进行对玩家进行强化->最后会有BOSS战斗
